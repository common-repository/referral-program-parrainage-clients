WLC Referral plugin allows you to add a referral program to your site using Wordpress & Woo Commerce. You MUST have a We Love Customers account to use this plugin.

== Description ==

WLC Referral plugin allows you to add a referral program to your site using Wordpress & Woo Commerce.
The referral program you create allows your visitors to share your website with their friends and get rewards if their friends completed a purchase or a specific action.
All the rules of the referral program have to be specified on the We Love Customers platform : content of messages for friends and customers, conditions and delay for getting reward...

== Installation ==

1. Create an account on We Love Customers using this page : https://www.welovecustomers.fr/solutions/parrainage-e-commerce/
2. Upload the full directory `wlc-referral` to the `/wp-content/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Configure the plugin through the specific menu

== Frequently asked questions ==

= Can I use the plugin without a We Love Customers account ?=
No, you can't. We Love Customers has develop a referral program solution. This solution needs you to configure an account and all the rules of you referral program using the We Love Customers platform.

= Why does the plugin communicate with We Love Customers servers ? =
The plugin needs to communicate with We Love Customers servers for some specific reasons :
- getting referral program configuration from our servers
- sending information about referrals and purchases to our servers
- creating on-the-fly discount coupons


= Why does the plugin call the Amazon CDN ? =
The plugin uses Amazon Cloudfront CDN for one main reason : the performance of their infrastructure!
Use of CDN allows you to automatically benefits from our performance our feature updates.
You can disabled the use of CDN from the settings.
