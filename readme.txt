Améliorez l'expérience client en créant votre programme de parrainage, un programme de fidélité, des enquêtes NPS et badge d'avis sur votre site WooCommerce grâce à la solution We Love Customers. Vous devez créer un compte sur la plateforme We Love Customers pour utiliser ce plugin.

== Description ==

Ce plugin vous permet de créer un programme de parrainage client, un programme de fidélité et de mettre en place des avis / enquêtes de satisfaction / enquêtes NPS sur votre site WooCommerce en quelques minutes. Vos clients pourront partager leurs achats et inviter leurs amis afin d'obtenir une récompense si leur filleul passe une commande. Personnalisez simplement votre programme ainsi que les messages pour vos clients et leurs filleuls...

== Installation ==

1. Créer un compte sur We Love Customers grâce à ce lien : https://www.welovecustomers.fr/solutions/parrainage-e-commerce/
2. Uploadez the répertoire `wlc-referral` dans le répertoire `/wp-content/` directory
3. Activaez le plugin depuis le menu 'Extensions' de WordPress
4. Configurez le menu à l'aide du menu parrainage

== Frequently asked questions ==

= Puis-je utiliser le plugin sans créer de compte sur We Love Customers ?=
Vous pouvez télécharger le plugin et l'installer, mais vous devrez créer un compte pour l'utiliser. Le plugin nécessite la création d'un compte sur la plateforme We Love Customers afin de paramétrer votre programme de parrainage et de bénéficier d'évolutions régulières.

= Pourquoi le plugin communique des données à We Love Customers ? =
Il y a plusieurs raisons :
- le plugin récupère les informations du programme de parrainage depuis nos serveurs
- le plugin envoie certaines informations d'achat pour valider les parrainages
- le plugin crée des coupons à la volée dans Wordpress WooCommerce en fonctions des parrainages


= Pourquoi le plugin utilise un CDN Amazon ? =
Le plugin utilise le CDN Cloudfront d'Amazon pour une raison : la performance de leur infrastructure !
Seules des données statiques (images, fichiers javascript ou CSS) y sont stockées.
